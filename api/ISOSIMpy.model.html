class ISOSIMpy.model.EPM_Unit(mtt, eta, bounds=None)[source]¶

Bases: object

Exponential piston flow model unit.

mtt¶

Mean travel time.

Type:

float

eta¶

Ratio of total volume to volume of exponential model (>= 1). eta = 1 means only exponential model, eta > 1 means exponential model with (eta - 1) part piston flow.

Type:

float

parameters¶

List of parameters.

Type:

list

bounds¶

The bounds for the parameters. Consists of a tuple with the structure (lower_bound, upper_bound) for each parameter.

Type:

list

class ISOSIMpy.model.Model(dt, lambda_, input_series, target_series, steady_state_input=None, n_warmup_half_lives=2)[source]¶

Bases: object

The model class.

dt¶

The time step size. Note that the units corresponding to the time step size must be consistent with all other time-dependent parameters / units. Has dimension [T]

Type:

float

lambda_¶

The decay constant with. Has dimension [1/T]

Type:

float

input_series¶

The input series. Can have arbitrary units, but must be consistent with the target series units.

Type:

array_like

target_series¶

The target series. Can have arbitrary units, but must be consistent with the input series units.

Type:

array_like

steady_state_input¶

The steady-state input. Can have arbitrary units, but must be consistent with the input and target series units. Defauls to None.

Type:

array_like, optional

n_warmup_half_lives¶

The number of half-lives to use for warmup. Defaults to 2.

Type:

int, optional

units¶

The list of model units (each item being an instance of a unit class).

Type:

list

parameters¶

The current parameters of the model. With n_units units, the last n_units parameters are the fractions of each unit. In the case of a single unit, the last parameter is 1.

Type:

array_like

unit_fractions¶

The fractions of each unit in the model; see parameters.

Type:

array_like

fixed_parameters¶

The parameters that are fixed during calibration. Is a list of bools (one for each parameter), where True indicates a fixed parameter.

Type:

array_like

bounds¶

The bounds for the parameters. Consists of a tuple with the structure (lower_bound, upper_bound) for each parameter, even if the parameter is fixed.

Type:

array_like

initial_parameters¶

The initial parameters of the model; see parameters.

Type:

array_like

model_is_warm¶

Whether the model has been warmed up or not.

Type:

bool

n_warmup¶

The number of warmup time steps derived from n_warmup_half_lives.

Type:

int

handle_fixed_parameters(parameters)[source]¶

Handle fixed parameters. This is done to conform with the differential evolution optimizer.

Parameters:

parameters (list) – The parameters to simulate the model with. With n_units units, the last n_units parameters are the fractions of each unit. In the case of 1 unit, the last parameter is 1.

Returns:

parameters_ – The parameters with fixed parameters removed.

Return type:

np.ndarray
